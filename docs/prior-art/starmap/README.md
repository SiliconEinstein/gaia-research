# Starmap Prior Art

This directory snapshots Gaia core's `gaia inspect starmap` visualization code
as reference material for future `gaia-research` graph-session visualization.

Gaia core still owns the active package-graph visualization primitive. This
copy is not shipped, imported, tested, or exposed by `gaia-research`.

Useful concepts to extract before implementing graph-session visualization:

- graph JSON shape for knowledge, strategy, and operator nodes;
- module clustering and cross-module edge treatment;
- DOT and SVG rendering paths;
- the Stellaris visual theme and SVG post-processing;
- browser template structure for interactive graph inspection.
