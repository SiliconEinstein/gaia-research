# Skill Prior Art

This directory archives research-adjacent Gaia skills that are useful design
input for `gaia-research`, but are not active packaged skills in this repository.

Status:

- not installed by `gaia-research`;
- not exposed through Gaia core `gaia skill register`;
- not part of runtime CI or mypy coverage;
- not compatibility contracts.

Promote material from here only by extracting a maintained `gaia-research`
workflow, CLI, SDK surface, or packaged skill with its own tests.
